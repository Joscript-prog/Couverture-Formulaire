# 📋 Couverture Formulaire - Audit 4G/5G

Application interne **IPKC** pour générer les rapports d'audit et d'intervention.

## Installation locale

1. Télécharge le projet
2. Copie le fichier `.env.example` et renomme-le en `.env`
3. Remplis ta clé Supabase dans le fichier `.env`
4. Ouvre le fichier `4g-5g/index.html`
